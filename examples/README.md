## 示例 Prompt

### 代碼解釋

```text
請用中文解釋下面這段代碼的作用、邊界情況和可能的 bug：

function sum(items) {
  return items.reduce((total, item) => total + item.price, 0)
}
```

```text
請把下面內容整理成 5 個要點，並列出 2 個值得追問的技術問題：

在這裡貼上文章片段。
```
