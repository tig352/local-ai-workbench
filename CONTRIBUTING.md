# Contributing

感謝你願意參與 Local AI Workbench。

## 適合貢獻的內容

- 模型 provider 適配建議
- 本地模型使用案例
- README 和文檔修正
- 小型 bug fix
- 可重現的問題回報

## 提交前檢查

- 不提交 `.env`、API key、token、cookie 或私有對話記錄
- 不提交 `node_modules`、`dist`、`build`
- PR 說明中寫清楚變更目的和測試結果
- 涉及資料存儲或網絡請求時，說明資料流向

## 開發原則

- 本地優先
- 權限最小化
- 不默認上傳用戶資料
- 小步迭代，可測試，可回滾
