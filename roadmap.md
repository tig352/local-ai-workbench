# Roadmap

## 兩週 MVP

### 第 1 週

- 明確資料模型：模型配置、prompt、執行記錄、導出格式
- 建立最小 Web UI 或 CLI 原型
- 支援 OpenAI compatible chat completions 調用
- 支援 Ollama endpoint 配置
- 完成 README、MVP 文檔和安全說明

狀態：已完成首版無依賴 Web 原型。

### 第 2 週

- Prompt 收藏與標籤
- 對同一 prompt 執行多模型比較
- 保存本地執行歷史
- 導出 Markdown / JSON
- 補充示例、截圖和基本測試

狀態：Prompt 收藏、多模型執行、歷史保存、Markdown / JSON 導出和基本測試已完成；下一步補真實模型連接截圖和使用案例。

## 1-2 個月

- 支援多 preset：摘要、翻譯、代碼解釋、資料抽取
- 支援簡單評分：長度、延遲、錯誤、人工評分
- 加入本地 SQLite 或文件存儲
- 增加示例工作流

## 3-4 個月

- 文檔站與 demo
- 批量 prompt 測試
- RAG mini lab 子模塊
- 導出可分享的模型測試報告

## 5-6 個月

- 1.0 版本穩定化
- 插件化模型 provider
- 更完整的測試與遷移工具
- 收集真實使用案例
